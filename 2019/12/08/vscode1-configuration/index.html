插件(体验更新ing)

python

必备插件python，提供了代码分析，高亮，规范化等很多基本功能

Anaconda Extension Pack

针对使用anaconda，可以大大增强代码提示功能。原始的代码提示基本只包含了python标准库，该插件可对各种第三方库基本都能实现代码提示了，并且还会额外显示每个方法的帮助。

c/c++

同上python，提供语言开发环境

code runner

Run code snippet or code file for multiple languages: C, C++, Java, JavaScript, PHP, Python, Perl, Perl 6, Ruby, Go, Lua, Groovy, PowerShell, BAT/CMD, BASH/SH, F# Script, F# (.NET Core), C# Script, C# (.NET Core), VBScript, TypeScript……

useges:

捷键方法常用

  • To run code:
    • use shortcut Ctrl+Alt+N
    • or press F1 and then select/type Run Code,
    • or right click the Text Editor and then click Run Code in editor context menu
    • or click Run Code button in editor title menu
  • To stop the running code:
    • use shortcut Ctrl+Alt+M
    • or press F1 and then select/type Stop Code Run
    • or right click the Output Channel and then click Stop Code Run in context menu
  • To select language to run, use shortcut Ctrl+Alt+J, or press F1 and then select/type Run By Language, then type or select the language to run: e.g php, javascript, bat, shellscript..

Configuration：

Ctrl+, then type code-runner
配置相应的编译/解释器：
Make sure the executor PATH of each language is set in the environment variable. You could also add entry into code-runner.executorMap to set the executor PATH

TODO Highlight

写代码过程中，突然发现一个Bug，但是又不想停下来手中的活，以免打断思路，怎么办？按代码编写会规范，都是建议在代码中加个TODO注释

如：

该插件作用：

  1. 分析代码中的TODO标签，并加亮之后，可以集中显示

  2. 可以list所有的todo位置

  3. 使用：

    命令面板中输入： Todohighlist 选择相应的命令

Settings Sync

多台设备之间，同步 VS Code 配置， 包括setting文件，插件设置等 。通过登录 GitHub 账号来使用这个同步工具

此外：vscode-syncing 也可多台设备之间，同步 VS Code 配置

  1. 下载插件
  2. log in github账号
  3. alt+shift+u 强制上传：会自动在github账号新建一个gist的token，然后上传成功后会返回gist id（会自动填充之配置文件，可以自己确定一下），保存
  4. 在其他设备上直接配置gist id
    然后 alt+shift+d 即可下载配置
